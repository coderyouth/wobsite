About CoderYouth

CoderYouth is a teenager-only community, that is, you can only register if you are under 20.

Why is this? In short, when you learn to code at a young age, your friends aren't interested, so in CoderYouth you can connect with others with the same interests as you.

If you want to know more, you should probably read this hackernoon article.
